# LANDROID
This is landroid a compilation of many termux fixes and installers to run linux(debian) on old android phones with mid to low end gpus and kernels like 4.14 this includes Sabrandif termux desktop anlinux debian and box64 from ryanfortner with many fixes to make an emulsification of these installers 
now mainly for minecraft superlite Android modpacks made by me
